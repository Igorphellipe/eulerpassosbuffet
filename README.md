# Euler Passos Buffet

Este projeto é um site institucional para o buffet Euler Passos, desenvolvido em HTML, CSS e JavaScript puro. O objetivo é apresentar os serviços oferecidos pelo buffet, facilitar o contato com clientes e permitir o gerenciamento de informações administrativas por meio de uma área restrita.

## Funcionalidades

- **Página inicial:** Apresentação do buffet, seus diferenciais e principais serviços.
- **Área administrativa:** Acesso restrito para gerenciamento de informações (admin.html).
- **Login:** Sistema simples de autenticação para acesso à área administrativa.
- **Listagem de serviços:** Exibição dinâmica dos serviços oferecidos, utilizando JavaScript.
- **Design responsivo:** Layout adaptado para diferentes dispositivos, proporcionando boa experiência em desktops e dispositivos móveis.

## Estrutura do Projeto

- `index.html`: Página principal do site.
- `admin.html`: Área administrativa para gerenciamento interno.
- `login.html`: Tela de login para acesso restrito.
- `assets/`: Imagens e outros recursos estáticos.
- `css/`: Arquivos de estilos (admin.css, login.css, style.css).
- `js/`: Scripts JavaScript para funcionalidades do site.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6)

## Como usar

1. Clone o repositório:
	```bash
	git clone https://github.com/seu-usuario/seu-repositorio.git
	```
2. Abra o arquivo `index.html` em seu navegador para acessar o site.
3. Para acessar a área administrativa, utilize a tela de login (`login.html`).
